# 可话花园

可话停运之后，很多人都把自己的记录、图片和情绪留在了导出包里。这个项目做的事情很简单：把那些原本只适合存档的数据，重新变成一个可以在浏览器里安静翻看的个人回忆网站。

它是一个纯前端静态站点，不依赖后端服务，也不会把你的数据上传到任何服务器。所有解析、读取和展示都只在你的浏览器里完成，适合长期保存，也适合偶尔回来看看过去的自己。

## 项目特点

- 完全本地运行，数据不会离开你的设备
- 支持直接导入可话导出文件夹，也支持读取压缩包
- 自动整理动态里的图片与视频，浏览更轻松
- 面向大体量数据做了优化，内容多时依然可以顺畅查看
- 可以直接部署到 GitHub Pages，作为长期可访问的静态网站保存

## 在线访问

项目部署完成后，可以直接通过下面这个地址访问：

[在线打开可话花园](https://jackmeds.github.io/kehua-memory-garden/)

如果你是第一次在手机上使用，推荐先把可话导出的文件夹压缩成一个 ZIP，再在网页里导入，这样兼容性和操作体验会更好。

## 本地运行

请先确保本机已经安装 Node.js 18 或更高版本，然后执行：

```bash
npm install
npm run dev
```

本地开发服务启动后，在浏览器打开提示的地址即可。

## 打包发布

如果你只想生成静态文件，可以执行：

```bash
npm run build
```

构建产物会输出到 `dist` 目录。仓库已经配置了自动部署流程，推送到主分支后会自动发布到 GitHub Pages。

## 适合谁

- 想长期保存可话导出内容的人
- 想把自己的回忆做成一个可访问静态页面的人
- 想在本地离线查看旧动态、图片和视频的人

## 技术说明

这个项目基于 React 和 Vite 构建，前端样式为原生样式表，适合继续定制界面、主题和交互。

## 版权与使用

项目代码采用宽松开源方式提供，你可以自行部署、修改和保存自己的版本。

如果你准备把它继续打磨成公开项目，也欢迎在此基础上继续扩展。
